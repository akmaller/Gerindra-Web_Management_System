1. Persiapan Lingkungan

  • PHP 8.3 dengan ekstensi: bcmath, ctype, curl, fileinfo, gd/imagick, json, mbstring, openssl, pdo_mysql, tokenizer, xml, zip.
  • Composer 2, Node.js 18+, npm.
  • MySQL/MariaDB & web server (Nginx/Apache).
  • Opsional: jpegoptim, optipng, pngquant, gifsicle, cwebp untuk optimasi gambar.

2. Clone Repository

git clone https://github.com/akmaller/Gerindra-Web_Management_System.git
cd Gerindra-Web_Management_System

3. Instal Dependensi

composer install --no-dev --optimize-autoloader
npm install && npm run build

4. Konfigurasi Lingkungan

cp .env.example .env
php artisan key:generate
php artisan storage:link

Edit file .env dan sesuaikan:

  • APP_URL, APP_NAME.
  • DB_DATABASE, DB_USERNAME, DB_PASSWORD.
  • Konfigurasi mail/storage tambahan jika diperlukan.

5. Migrasi & Seeder

php artisan migrate --force
php artisan db:seed

Seeder akan memuat:

  • Identitas situs & company profile.
  • Berita awal lengkap dengan kategori, tag, dan struktur menu Gerindra â†’ Profil (menu otomatis muncul ketika halaman profil sudah dibuat manual).

6. Membuat Pengguna Admin

php artisan make:filament-user

Akses panel admin melalui /admin kemudian login menggunakan akun yang dibuat.

7. Menjalankan Aplikasi

php artisan serve

Buka http://127.0.0.1:8000 untuk melihat frontend, dan /admin untuk panel Filament.

8. Produksi & Deploy

  • Pastikan direktori storage dan bootstrap/cache writable.
  • Jalankan perintah optimasi: php artisan optimize, php artisan optimize:clear bila perlu.
  • Contoh konfigurasi Nginx:
server {
    server_name gerindra.id;
    root /var/www/gerindra/public;

    index index.php index.html;
    client_max_body_size 16m;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/run/php/php8.3-fpm.sock;
    }
}
